Developer Tools

Each tool is focused on one task and does it well

Reads git diff --staged and drafts a conventional commit message

$ git add src/auth.go
$ bonsai tool commit
feat(auth): add JWT token validation middleware

Reads your branch commit log and generates a structured PR description

$ bonsai tool pr
## Title
Add JWT authentication middleware
 
## Summary
- Add JWT validation for protected API routes
- Extract user claims into request context
- Add token refresh endpoint
 
## Test plan
- Verify valid tokens are accepted
- Verify expired tokens are rejected

Ask any coding question and get a concise, direct answer

$ bonsai tool ask "what does defer do in Go?"
defer schedules a function call to run after the
surrounding function returns. Deferred calls are
executed in LIFO order. Common uses: closing files,
unlocking mutexes, and cleanup operations.

Explain errors and stack traces with actionable fix directions

$ bonsai tool error "ECONNREFUSED 127.0.0.1:5432"
Your app tried to connect to PostgreSQL on
localhost:5432 but no server is listening there.
Start your database or check the connection config.

Describe what you want to match; get a regex with explanation

$ bonsai tool regex "ISO 8601 date"
Pattern: ^\d{4}-(?:0[1-9]|1[0-2])-(?:0[1-9]|[12]\d|3[01])$
Explanation: Matches dates in YYYY-MM-DD format
Examples: 2026-04-03, 2025-12-31

Pipe a function in, get a Google-style docstring back

$ cat auth.py | bonsai tool docstring
"""Validate a JWT token and return decoded claims.
 
Args:
  token: The JWT string to validate.
  secret: The signing key for verification.
 
Returns:
  A dict of decoded claims if valid.
 
Raises:
  InvalidTokenError: If the token is expired or malformed.
"""

2

Install Bonsai

Pull a model and go

bonsai pull bonsai-4b && bonsai tool ask "hello"
